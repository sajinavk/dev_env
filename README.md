## Quick Install

You can set up the environment with either method below:

### Option 1: Download and run
```bash
wget https://raw.githubusercontent.com/MichaelWoodc/dev_env/main/env_setup.sh -O env_setup.sh
chmod +x env_setup.sh
sudo ./env_setup.sh

```
